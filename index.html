Herramientas de Visualización

La etapa de visualización es la que finalmente concluye el proceso de Business Intelligence , el cual tiene como objetivo convertir datos en información para posterior pasar a Business Analitycs (modelos predictivos), de la información en conocimiento, para finalmente aplicar Data Science (modelos prescriptivos) y caminar del conocimiento a la sabiduria ., la visualización es la etapa final de todo un proceso, este si es el petroleo de los datos

Ankerst define la minería de datos visual como: “… un paso en el proceso de descubrimiento del conocimiento que utiliza la visualización como canal de comunicaciones entre la computadora y el usuario para obtener patrones nuevos e interpretables” [Ankerst 99]

El presente dashboard muestra resultados de un estudio en particular, los datos son una pequena muestra para fines educativos, se utilizo la metodologia KDD y se aplicaron técnicas de machine learning k;means y random forest para clasificar a los clientes en clusters para identificar patrones.
